# PWA Icons

This directory should contain the following icon files for the Progressive Web App:

- `icon-192x192.png`: 192x192 pixel icon for Android devices
- `icon-512x512.png`: 512x512 pixel icon for Android devices
- `apple-touch-icon.png`: 180x180 pixel icon for iOS devices

These icons are referenced in the web app manifest and are required for proper PWA installation on devices.

## Icon Requirements

- All icons should be in PNG format
- Icons should have a transparent background
- Icons should be properly sized (exact dimensions as specified)
- Icons should be visually appealing at their respective sizes

## How to Generate Icons

You can generate these icons from a master SVG or high-resolution PNG using tools like:

- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [Favicon Generator](https://realfavicongenerator.net/)
- Adobe Photoshop or Illustrator
- GIMP or Inkscape (free alternatives)

## Implementation Note

For the Talemo project, these icons should feature the Talemo logo with appropriate padding and should match the brand colors defined in the design system.
